# Governance Standards

Security, compliance, data governance, and architecture decision standards for all projects.

## Standards Index

| Standard | Description |
| -------- | ----------- |
| [Security Guidelines](./security-guidelines) | OWASP compliance, input validation, secrets management |
| [Data Governance & Privacy](./data-governance) | GDPR/CCPA compliance, PII handling, data retention |
| [Feature Flags](./feature-flags) | Progressive rollout, canary deployments, kill switches |
| [Documentation Standards](./documentation-standards) | README templates, API docs, code comments, changelogs |
| [Accessibility Standards](./accessibility-standards) | WCAG 2.1 AA compliance, testing procedures |
| [Database Standards](./database-standards) | Schema design, indexing, migrations, query optimization |
| [ADR Template](./adr-template) | Architecture Decision Records template and guidelines |
| [Infrastructure Security](./infrastructure-security) | Server hardening, firewall, SSH, TLS, container security |
| [Self-Hosting Standards](./self-hosting-standards) | Reverse proxy, process management, DNS, self-hosted infrastructure |
| [Open Source](./open-source) | Licensing, repository health files, sensitive data protection, community standards |
